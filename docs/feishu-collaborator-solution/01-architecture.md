# 方案架构

## 整体架构

```
┌─────────────────────────────────────────────────────────┐
│                        用户                              │
│                   点击飞书链接                            │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│                   飞书文档/多维表格                       │
│                 （用户无权限，显示申请页面）               │
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │  用户填写申请信息：                               │   │
│  │  • 姓名/备注：XXX                                 │   │
│  │  • 申请理由：【卡密】ABC-1234-5678-XYZ            │   │
│  └─────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│                   飞书开放平台                           │
│              （权限申请事件回调）                         │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│                   机器人自动处理                         │
│                                                         │
│  1. 接收权限申请事件                                    │
│  2. 从申请理由中提取卡密                                 │
│  3. 查询多维表格验证卡密                                 │
│  4. 验证通过 → 添加为协作者                              │
│  5. 验证失败 → 拒绝申请 + 通知用户                       │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│                   飞书多维表格                           │
│                  （存储卡密和日志）                       │
└─────────────────────────────────────────────────────────┘
```

## 核心流程

### 用户访问流程

```
用户点击飞书链接
    │
    ▼
┌─────────────────────────────────────────────┐
│  飞书文档打开                                │
│  （用户无权限）                              │
│                                             │
│  ┌─────────────────────────────────────┐    │
│  │  🔒 您没有访问权限                    │    │
│  │                                     │    │
│  │  [申请访问权限]                      │    │
│  └─────────────────────────────────────┘    │
└─────────────────────────────────────────────┘
    │
    ▼ 用户点击申请
┌─────────────────────────────────────────────┐
│  申请访问权限                                │
│                                             │
│  申请理由：【卡密】ABC-1234-5678-XYZ         │
│                                             │
│  [提交申请]                                  │
└─────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────┐
│  机器人自动处理                              │
│                                             │
│  ✅ 卡密有效 → 自动通过                      │
│  ❌ 卡密无效 → 自动拒绝 + 提示原因           │
└─────────────────────────────────────────────┘
    │
    ▼ 成功
┌─────────────────────────────────────────────┐
│  ✅ 您已获得访问权限                         │
│  即可查看文档内容                            │
└─────────────────────────────────────────────┘
```

### 验证逻辑流程

```
收到权限申请事件
    │
    ▼
┌─────────────────────┐
│ 提取申请理由         │
│ 解析卡密            │
└─────────────────────┘
    │
    ▼
┌─────────────────────┐
│ 查询卡密表          │
│ 验证卡密有效性      │
└─────────────────────┘
    │
    ├── 不存在 ──▶ 拒绝申请，通知"卡密不存在"
    │
    ├── 已停用 ──▶ 拒绝申请，通知"卡密已停用"
    │
    ├── 已过期 ──▶ 拒绝申请，通知"卡密已过期"
    │
    ├── 已被使用 ──▶ 拒绝申请，通知"卡密已被使用"
    │
    ▼ 有效
┌─────────────────────┐
│ 添加用户为协作者     │
│ 记录验证日志        │
│ 更新卡密状态        │
└─────────────────────┘
    │
    ▼
┌─────────────────────┐
│ 通知用户验证成功     │
└─────────────────────┘
```

## 技术组件

### 飞书开放平台 API

| API | 用途 |
|-----|------|
| `drive.permission.member.create` | 添加协作者 |
| `drive.permission.member.delete` | 移除协作者 |
| `bitable:record:read` | 读取多维表格卡密数据 |
| `bitable:record:write` | 写入验证日志 |

### 事件订阅

| 事件 | 说明 |
|------|------|
| `drive.permission.apply_event_v1` | 文档权限申请事件 |

## 部署方式

飞书机器人支持两种部署方式：

### 方式1：飞书云函数（推荐）

```
┌─────────────────────────────────────────────────────────┐
│                   飞书云函数                             │
│                                                         │
│  • 无需自己部署服务器                                    │
│  • 飞书自动触发                                          │
│  • 免费额度充足                                          │
└─────────────────────────────────────────────────────────┘
```

### 方式2：自建服务器

```
┌─────────────────────────────────────────────────────────┐
│                   你的服务器                             │
│                                                         │
│  • 需要公网可访问的地址                                  │
│  • 接收飞书 Webhook 回调                                 │
│  • 可使用 Serverless 平台（如 Cloudflare Workers）       │
└─────────────────────────────────────────────────────────┘
```
