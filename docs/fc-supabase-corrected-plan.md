## 一、阿里云 FC 真实免费政策

### 1.1 官方政策（已更正）

根据阿里云官方文档：https://help.aliyun.com/zh/functioncompute/fc/product-overview/trial-quota-1

| 项目 | 免费额度 | 说明 |
|------|----------|------|
| **试用时长** | **3 个月** | 新用户首次开通 |
| **试用 CU** | **15 万 CU** | 计算资源单位 |
| **试用后** | **按量付费** | 无长期免费额度 |

### 1.2 CU 计算方式

```
1 CU = 1 GB-秒（1秒 × 1GB内存）

你的用量计算：
├── 每次访问执行时间：约 500ms（0.5秒）
├── 内存配置：512MB（0.5GB）
├── 每次调用消耗：0.5秒 × 0.5GB = 0.25 CU
├── 每天 100-300 次访问：25-75 CU/天
├── 每月消耗：750-2250 CU/月
└── 15万 CU 可用：约 67-200 个月（理论值）

⚠️ 但试用期只有 3 个月！
```

### 1.3 试用期满后的费用

| 资源类型 | 单价 | 月用量 | 月费用 |
|----------|------|--------|--------|
| 函数调用次数 | ¥0.0133/万次 | 6000 次 | ¥0.008 |
| 执行时长 | ¥0.00003167/CU | 2250 CU | ¥0.07 |
| 公网流量 | ¥0.5/GB | 0.5 GB | ¥0.25 |
| **月总计** | - | - | **约 ¥0.33** |

**结论**：即使试用期结束，按你的用量，月费用也不到 1 元！
