## 一、问题描述

### 现象
在 iframe 中嵌入飞书多维表格应用模式时，页面底部出现「进入原应用」按钮，用户点击后可跳转到飞书原生应用，影响卡密验证系统的封闭性。

### 官方声明
> 目前 iframe 形式嵌套多维表格支持不完善，可能会出现不兼容的 bug，后续可能会有调整。请知悉可能出现的问题，谨慎使用。
> 
> —— 来源：飞书内部技术文档

### 根因分析
根据飞书产品逻辑，按钮出现与以下因素相关：
1. **飞书版本更新**：2025年底至2026年初全量开放应用模式后新增此引导按钮
2. **权限配置变更**：分享范围、角色权限变化可能触发按钮显示
3. **安全策略限制**：域名未加白名单、iframe sandbox 属性限制等
4. **用户身份变化**：未登录或身份切换

## 二、当前解决方案

### 方案1：URL参数测试（2026-03-13 已验证无效）

#### 测试的URL参数
以下参数均已测试，均无法隐藏「进入原应用」按钮：

| 参数 | 页面访问 | 按钮状态 |
|------|----------|----------|
| `?embed=1` | ✅ 正常 | ❌ 依然存在 |
| `?hide_bottom=1` | ✅ 正常 | ❌ 依然存在 |
| `?hide_back_button=1` | ✅ 正常 | ❌ 依然存在 |
| `?embedded=1` | ✅ 正常 | ❌ 依然存在 |
| `?header=false` | ✅ 正常 | ❌ 依然存在 |
| `?nav=false` | ✅ 正常 | ❌ 依然存在 |

**结论**：飞书多维表格不支持通过URL参数隐藏UI元素，所有参数对「进入原应用」按钮无效。

### 方案2：CSS 裁剪底部区域（2026-03-13 已验证有效）

#### 实现原理
通过CSS让iframe高度超出容器，使底部按钮区域被裁剪到容器外，实现视觉隐藏。

#### 代码实现
```css
/* src/static/index.html */

.iframe-container {
    flex: 1;
    position: relative;
    overflow: hidden;  /* 关键：隐藏超出容器的内容 */
    width: 100%;
}

.iframe-container .feishu-frame {
    height: calc(100% + 50px);  /* 高度增加50px，底部被裁剪 */
    width: 100%;
    position: absolute;
    top: 0;   /* 从顶部定位，让底部超出 */
    left: 0;
}
```

#### 裁剪高度测试结果
| 裁剪高度 | 效果 |
|----------|------|
| 40px | ❌ 按钮露出一点 |
| 50px | ✅ 完全隐藏 |
| 60px | ✅ 隐藏（略微过度） |
| 80px | ✅ 隐藏（过度裁剪） |

**最佳值**：`height: calc(100% + 50px)`

#### 优点
- 无侵入式，不修改飞书页面内容
- 兼容性好，纯CSS实现
- 不影响页面其他功能

#### 注意事项
- 修改CSS后需要强制刷新浏览器（`Ctrl + Shift + R`）清除缓存
- 底部其他内容也会被裁剪（如有）

**状态**：✅ 当前使用，效果良好

#### 防御性方案（应对按钮位置变化）

由于浏览器跨域限制，无法通过JavaScript检测iframe内部元素位置。采用以下防御措施：

```css
/* 顶部右侧遮罩 - 防止按钮移到右上角 */
.feishu-top-right-mask {
    position: absolute;
    top: 0;
    right: 0;
    width: 220px;
    height: 80px;
    background: transparent;
    z-index: 9999;
    pointer-events: auto;
}
```

**覆盖区域**：
| 位置 | 防御措施 | 覆盖范围 |
|------|----------|----------|
| 底部 | CSS裁剪 | 底部50px |
| 右上角 | 透明遮罩 | 220px × 80px |

**监测机制**：
- 每周定期人工检查按钮位置是否变化
- 用户反馈异常时及时响应
- 飞书更新后立即验证

### 方案3：CSS 遮罩层（已弃用）
```css
/* src/static/index.html */

/* 顶部右侧遮罩 */
.feishu-top-right-mask {
    width: 220px;
    height: 80px;
    z-index: 9999;
}

/* 底部遮罩 - 覆盖"进入原应用"按钮 */
.feishu-bottom-mask {
    height: 60px;
    z-index: 9999;
    pointer-events: auto;
}
```

**状态**：❌ 已弃用，替换为方案2（CSS裁剪）
