## 五、优化建议

### 5.1 访问日志清理策略

如果担心访问日志增长过快，可以设置定期清理：

```sql
-- 删除 90 天前的访问日志
DELETE FROM access_logs WHERE access_time < NOW() - INTERVAL '90 days';

-- 或者保留最近 10000 条
DELETE FROM access_logs WHERE id NOT IN (
    SELECT id FROM access_logs ORDER BY access_time DESC LIMIT 10000
);
```

### 5.2 监控存储使用

```sql
-- 查看各表大小
SELECT 
    tablename,
    pg_size_pretty(pg_total_relation_size(tablename::text)) AS size
FROM pg_tables 
WHERE schemaname = 'public'
ORDER BY pg_total_relation_size(tablename::text) DESC;
```
