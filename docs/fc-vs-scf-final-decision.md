## 二、核心维度对比

### 2.1 价格（试用期后）

| 对比项 | 阿里云 FC | 腾讯云 SCF |
|--------|-----------|------------|
| 试用期 | 3 个月 | 3 个月 |
| 调用费用 | ¥0.0133/万次 | ¥0.0133/万次 |
| 执行费用 | ¥0.00003167/GB-秒 | ¥0.00011108/GB-秒 |
| 你的月费用 | ~¥0.33 | ~¥0.35 |
| **价格结论** | **略便宜** | 略贵 |

**计算详情**：
```
你的月用量：6000次调用 + 2250 GB-秒

阿里云 FC：
- 调用费：6000 × 0.0133/10000 = ¥0.008
- 执行费：2250 × 0.00003167 = ¥0.07
- 流量费：0.5GB × ¥0.5 = ¥0.25
- 合计：约 ¥0.33/月

腾讯云 SCF：
- 调用费：6000 × 0.0133/10000 = ¥0.008
- 执行费：2250 × 0.00011108 = ¥0.25
- 流量费：0.5GB × ¥0.8 = ¥0.4
- 合计：约 ¥0.66/月
```

**注意**：腾讯云的执行时长单价略高，但实际差异很小。

### 推荐结论

## 🏆 推荐：阿里云 FC

**理由**：

| 优势 | 说明 |
|------|------|
| 1️⃣ **文档更全面** | 遇到问题更容易找到解决方案 |
| 2️⃣ **社区更活跃** | Python/FastAPI 相关问题搜索结果多 |
| 3️⃣ **FastAPI 支持更好** | 官方有现成的适配示例 |
| 4️⃣ **工具链更现代** | `s` 工具体验更好 |
| 5️⃣ **价格略低** | 虽然差异很小 |

**差距分析**：
- 腾讯云 SCF 也很优秀，但文档和社区资源略少
- 对于个人开发者，文档质量和社区支持很重要
- 阿里云 FC 的优势在这些方面更明显

## 五、选择建议

### 选择阿里云 FC 的情况 ✅
