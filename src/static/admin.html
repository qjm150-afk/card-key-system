导入销售状态

下载CSV模板，填写卡密、订单号、销售状态和销售渠道后上传

支持格式：CSV
销售状态可选值：未售出、已售出、已退款、有纠纷
销售渠道可选值：阿奇索发货平台、小红书、淘宝、抖音、拼多多、闲鱼、微信私域 或自定义
注意：已退款/有纠纷的卡密将自动停用

导入卡密信息

下载CSV模板，填写卡密信息后上传

支持字段：
卡密值（必填）、激活状态、过期时间、备注、链接名称、销售状态、销售渠道、订单号、访问密码、飞书链接、最大设备数

字段说明：
• 激活状态：有效 / 已停用（"已激活"由系统自动判断，不可导入）
• 销售状态：未售出 / 已售出 / 已退款 / 有纠纷
• 过期时间格式：2026-12-31 23:59:59

✓ 卡密已存在：更新该卡密信息
✓ 卡密不存在：新增卡密记录
○ 空值字段：保留原值不更新
